# Requirements:
- Linux or OS X
- C++11
- Python3
- GNU Make

# How to build and run

## Coordinator:
```bash
$ make run-coordinator
```

## Process (all case studies):
```bash
$ make run-process
```

## Case N=0,1,2,3
```bash
$ make run-caseN
```

# Verify files
## Logs:
```bash
$ make verify-logs
```
